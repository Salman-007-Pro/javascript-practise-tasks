Code Challenge # 4

Q: Let remember the first coding challenge where Mark and John compare their BMI. Let now implements the same functionality with objects and methods.

  • 1. For each of them, create an object with properties for their full name, mass, and height.
  • 2. Then, add a method to each object to calculate the BMI. Save the BMI to the object and also return if from the method.
  • 3. In the end, log to the console who has the highest BMI, together with the full name and the respective BMI. Donot forget ther might have the same BMI.
  • Remember: BMI = Mass / height * 2. (mass in kg and height in meter).
